How It Works

OpenVault uses semantic camouflage to hide encrypted data inside files that look completely normal. Your secrets are encrypted with AES-256-GCM and embedded as invisible Unicode characters.

Security
  • AES-256-GCM encryption
  • 119-bit entropy passwords
  • Zero-width character steganography
  • No server or cloud dependency
  • Everything happens on your device
